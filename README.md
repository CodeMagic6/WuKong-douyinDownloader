# WuKong 抖音下载

开箱即用的抖音视频下载工具。粘贴链接自动下载，支持合集批量提取。

## 快速上手

### 1. 下载项目

```bash
git clone https://github.com/CodeMagic6/WuKong-douyinDownloader.git
cd WuKong-douyinDownloader
npm install
```

### 2. 运行程序

```bash
node server.js
```

浏览器自动打开 http://localhost:3000

### 3. 扫码登录

工具需要登录才能下载。

1. 在页面顶部点击 **登录: 未登录 → 扫码登录**
2. 自动弹出浏览器窗口，显示抖音登录页
3. 用抖音 App 扫码，或手机号登录
4. 弹窗自动关闭，状态变为 **已登录**

> 下次使用无需重复登录，Cookie 自动保存。

### 4. 下载视频

**手动粘贴**: 复制抖音视频链接，粘贴到输入框，点击"添加下载"

**自动捕获**: 设置面板开启 **剪贴板捕获** → 复制链接自动填入地址栏。开启 **捕获后自动下载** → 复制即下载。

## 功能

| 功能 | 说明 |
|------|------|
| 扫码登录 | 点一下，扫码即登录，无需手动设置 |
| 单视频下载 | 粘贴链接自动下 |
| 合集批量下载 | 自动翻页扫描全部视频 |
| 剪贴板捕获 | 复制链接自动识别 |
| 实时进度 | 速度、ETA、百分比 |
| 并发控制 | 同时下 1-10 个 |
| 下载目录 | 自定义保存位置 |

## 常见问题

**页面显示"未登录"？** 点击登录状态 → 扫码登录即可。

**下载失败？** 检查网络，工具会自动重试。

**服务断开？** Windows 下点击控制台窗口会暂停进程。重启工具即可。
